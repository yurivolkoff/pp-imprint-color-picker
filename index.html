v4 changes
  • Sections reordered — Imprint colors → Artwork file → Artwork instructions. Colors first since they're the structured-data input the artwork team most depends on; file upload is optional and can come after the order.
  • Artwork file got a section title (was the only section without one).
  • Typography system defined as CSS custom properties + helper classes. Three section titles ("Imprint colors", "Artwork file", "Artwork instructions") all share the same style: 14 px / 700 / normal-case / text-dark. Sub-labels (context labels, summary card, demo rows) all share 11 px / 700 / uppercase / text-muted. Help text consistent at 12 px / 400 / text-muted.
  • Button system — .btn-primary (orange, modal commit), .btn-secondary (navy, picker done), .btn-outline (add color closed), .btn-toggle (add color when picker open, navy filled), .btn-ghost (close, foot links), .btn-pill-sm (PMS filters, demo buttons). All use the same 0.15 s ease transition; hover patterns are consistent within each variant.
  • Standard palette layout — label-left, swatches-right grid. Each family is one row (or wraps if needed). Recently-used row removed. Vertical height down from ~360 px to ~220 px.
  • Picker is now an overlay within the modal body — when open it absolute-positions over the artwork sections instead of pushing them down. Modal height stays fixed at 640 px. Sections behind the picker fade to 18% opacity for focus.
  • Auto-scroll on open — chip-zone scrolls to the top of the modal viewport when the picker opens, so chips stay visible above the picker.
  • Stable scrollbar gutter (scrollbar-gutter: stable) — modal-body reserves space for the scrollbar always, so content doesn't shift horizontally when the scrollbar appears.
